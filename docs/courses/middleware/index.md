# 中间件快速入门

将具体业务和底层逻辑解耦的组件，非业务的技术类组件非业务的技术类组件。是一种应用于分布式系统的基础软件，位于应用与操作系统、数据库之间，为上层应用软件提供开发、运行和集成的平台。中间件解决了异构网络环境下软件互联和互操作等共性问题，并提供标准接口、协议，为应用软件间共享资源提供了可复用的“标准件”。
